# ⚡ RapidFetch

Un sistema de información tipo neofetch pero moderno, rápido y personalizable.

## 🚀 Características

- 🎨 Modo gradient
- 🌙 Modo dark
- 🧠 Información del sistema (RAM, CPU, Disco)
- 🏠 Versión personalizada (rapidhomefetch)
- 🔥 Nombre del usuario en ASCII

## 📦 Instalación

```bash
pipx install rapidfetch

# Forma de Uso

## Simple
rapidfetch
rapidfetch --gradient
rapidfetch --dark
